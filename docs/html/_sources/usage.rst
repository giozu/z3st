Usage
=====

To run a Z3ST simulation:

.. code-block:: bash

    python z3st.py

Optional flags:
    --debug       Enable verbose solver information
    --mesh_plot   Display the generated mesh before solving

Example case folder structure:

.. code-block:: text

    case/
    ├── input.yaml
    ├── geometry/
    ├── mesh/
    ├── materials/
    └── output/
