Contributing
============

We welcome contributions to improve Z3ST!

1. Fork the repository:
   https://github.com/giozu/z3st

2. Create a new branch:
   .. code-block:: bash

        git checkout -b feature/my-improvement

3. Commit and push your changes.

4. Open a pull request describing your update.

Please follow the coding conventions (PEP8) and include docstrings in Google or NumPy format.
